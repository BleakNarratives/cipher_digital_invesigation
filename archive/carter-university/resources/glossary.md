## A

**Address (Crypto)**  
A unique identifier for a cryptocurrency wallet. Like a bank account number, but public.  
*Example*: `0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb`

**API (Application Programming Interface)**  
A way for programs to talk to each other. Like a waiter taking your order to the kitchen.

**Audit Log**  
A record of all actions taken. Used to track who did what and when.

## B

**Bash**  
A command language for Unix/Linux terminals. The language you type commands in.

**Bitcoin (BTC)**  
The first cryptocurrency, created in 2009. Digital gold.

**Blockchain**  
A public ledger that records all cryptocurrency transactions. Like a notebook everyone can read but no one can erase.

**Brute Force**  
Trying every possible combination until you find the right one. Like trying every key on a keyring.

## C

**Case Management**  
Organizing and tracking investigation details, evidence, and findings.

**Chain of Custody**  
Documentation showing who handled evidence and when. Critical for legal cases.

**CLI (Command Line Interface)**  
Text-based way to interact with a computer. The terminal.

**Clustering**  
Grouping related addresses together. Helps identify who owns multiple wallets.

**Command**  
An instruction you type in the terminal. Like `ls` or `cd`.

**Cryptocurrency**  
Digital money secured by cryptography. Bitcoin, Ethereum, etc.

## D

**Dashboard**  
A visual interface showing important information at a glance.

**Directory**  
A folder. Contains files and other directories.

**DeFi (Decentralized Finance)**  
Financial services using cryptocurrency instead of banks.

## E

**Encryption**  
Scrambling data so only authorized people can read it. Like a secret code.

**Ethereum (ETH)**  
A cryptocurrency platform that can run smart contracts. Created in 2015.

**Evidence**  
Information used to prove or disprove something in an investigation.

**Explorer (Blockchain)**  
A website where you can view blockchain transactions. Like Etherscan.io.

## F

**Forensics**  
The scientific method of gathering and analyzing evidence.

**FOSS (Free and Open Source Software)**  
Software that's free to use and anyone can see the code.

## G

**Gas Fee**  
The cost to process a transaction on Ethereum. Like a transaction fee.

**Grep**  
A command to search for text in files. Very powerful for investigators.

**GUI (Graphical User Interface)**  
Point-and-click interface. Opposite of CLI.

## H

**Hash**  
A unique fingerprint of data. Same input always gives same hash.  
*Example*: SHA-256 hash of "hello" is always `2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824`

**HMAC**  
Hash-based Message Authentication Code. Proves data hasn't been tampered with.

## I

**IP Address**  
Internet Protocol address. Your computer's address on the internet.  
*Example*: `192.168.1.1`

**Investigation**  
Systematic examination to discover facts.

## J

**JSON**  
JavaScript Object Notation. A way to structure data.  
*Example*: `{"name": "Calvin", "role": "investigator"}`

## K

**Key Pair**  
Public key (shareable) and private key (secret). Used in cryptography.

## L

**Ledger**  
A record of transactions. Blockchain is a public ledger.

**Log**  
A record of events. Like a diary for computers.

## M

**Mnemonic**  
A list of words used to recover a crypto wallet. Usually 12 or 24 words.  
*Example*: `abandon abandon abandon abandon abandon abandon abandon abandon abandon abandon abandon about`

**Mixer**  
A service that obscures the source of cryptocurrency. Used by criminals.

**ML (Machine Learning)**  
Teaching computers to learn patterns from data.

## N

**Node**  
A computer that participates in a blockchain network.

**NPM (Node Package Manager)**  
Tool for installing JavaScript packages. Like an app store for code.

## O

**OSINT (Open Source Intelligence)**  
Gathering information from publicly available sources.

**Output**  
The result of a command or program.

## P

**Path**  
The location of a file or folder.  
*Example*: `/Users/Calvin/Documents/case-001.txt`

**Peel Chain**  
A pattern where funds are split repeatedly. Common in money laundering.

**Pipe (`|`)**  
Sends output of one command as input to another.  
*Example*: `cat file.txt | grep "search"`

**Private Key**  
Secret code that proves you own a crypto wallet. NEVER share this!

**Public Key**  
Shareable part of a key pair. Derives your wallet address.

## Q

**Query**  
A request for information. Like asking a database a question.

## R

**Rate Limiting**  
Restricting how many requests someone can make. Prevents abuse.

**Redirection (`>`, `>>`)**  
Sending command output to a file instead of screen.

**RPC (Remote Procedure Call)**  
A way to interact with blockchain nodes.

## S

**Sandbox**  
A safe environment for testing. Can't harm real systems.

**Script**  
A file containing commands to run automatically.

**SHA-256**  
A cryptographic hash function. Creates 256-bit fingerprints.

**Smart Contract**  
Self-executing code on a blockchain. Like a vending machine.

**Solana (SOL)**  
A fast cryptocurrency platform.

## T

**Terminal**  
Text-based interface for controlling your computer.

**Token**  
A cryptocurrency built on another blockchain (like Ethereum).

**Transaction**  
Transfer of cryptocurrency from one address to another.

**Tx Hash**  
Unique identifier for a transaction.  
*Example*: `0x123abc...`

## U

**Unix**  
An operating system. Linux and Mac are Unix-like.

**URL**  
Web address.  
*Example*: `https://etherscan.io`

## V

**Variable**  
A named storage location for data.  
*Example*: `name="Calvin"`

**Verification**  
Confirming something is true or valid.

## W

**Wallet**  
Software that stores your cryptocurrency keys.

**Wildcard (`*`)**  
Matches any characters.  
*Example*: `*.txt` matches all text files

**Worker Thread**  
A separate process that runs in parallel. Makes programs faster.

## X

**XSS (Cross-Site Scripting)**  
A type of security attack. Injecting malicious code into websites.

## Z

**Zero-Day**  
A security vulnerability that's unknown to the software maker.

**Zod**  
A TypeScript library for data validation.
